# Latihan 01 — Issue Pertama

**Tujuan:** Buat issue dari template dan assign ke diri sendiri.

## Checklist

- [ ] Buka repo: https://github.com/fiidev/git-sandbox
- [ ] Klik tab **Issues** → **New issue**
- [ ] Pilih template **Latihan**
- [ ] Isi judul, misalnya: `latihan: eksplorasi workflow issue`
- [ ] Centang checklist di body issue
- [ ] Submit issue
- [ ] Assign issue ke akun **fiidev** (Assignees → yourself)
- [ ] Tambahkan label `latihan` jika belum otomatis

## Via CLI (alternatif)

```bash
gh issue create \
  --title "latihan: eksplorasi workflow issue" \
  --body "Latihan pertama — belajar buat issue dari CLI" \
  --label "latihan"
```

## Verifikasi

Issue muncul di tab Issues dengan assignee dan label yang benar.
